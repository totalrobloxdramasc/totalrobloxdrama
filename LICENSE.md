## 🛠️ Troubleshooting Common Issues

**❌ GUI doesn't appear after execution**
The overlay failed to render, usually due to a UI layer conflict.
**Solution:** Disable any other active scripts first, then re-execute the **Total Roblox Drama Script**. If the issue persists, reset your executor's render settings and try again.

**❌ Script crashes immediately on Android**
Executor version mismatch or insufficient device memory.
**Solution:** Update your mobile executor `.apk` to the latest release and close all background applications before launching. Ensure your device runs Android 8.0 or higher.

**❌ Auto Farm stops working mid-session**
The game's map may have been updated, changing resource node positions.
**Solution:** Download the latest **Best Total Roblox Drama Script** release, which includes updated node coordinates for the current map version.

**❌ Executor fails to attach to Roblox process**
Often caused by missing admin privileges or antivirus interference.
**Solution:** Right-click your executor and select **Run as Administrator**. Add the executor folder to your antivirus exclusion list at `C:/Desktop/TotalDrama/`.

**❌ Speed hack causes immediate server kick**
Speed multiplier set too high for the server's anti-cheat threshold.
**Solution:** Lower `speed_multiplier` to `1.5` or `2.0` and enable **Safe Mode** in the config settings.

**❌ iOS trust certificate revoked**
Apple periodically revokes sideloaded developer certificates.
**Solution:** Re-download the executor `.ipa` and reinstall it via your sideloading method. Trust the new certificate in `Settings > General > VPN & Device Management`.
